---
title: Quickstart
description: Get the Energy Contracts Dashboard running locally in under five minutes.
---

<Note>
  The Energy Contracts Dashboard requires a [Supabase](https://supabase.com) account for database storage and user authentication. Create a free project before proceeding.
</Note>

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/your-org/energy-contract-dashboard.git
    cd energy-contract-dashboard
    ```
  </Step>

  <Step title="Set up the server">
    Navigate to the server directory and install Python dependencies.

    ```bash
    cd server
    ```

    <CodeGroup>
      ```bash pip3
      pip3 install -r requirements.txt
      ```

      ```bash pip
      pip install -r requirements.txt
      ```
    </CodeGroup>

    Copy the example environment file and fill in your Supabase credentials:

    ```bash
    cp .env.example .env
    ```

    Open `.env` and set your values:

    ```env .env
    SUPABASE_URL=your_supabase_project_url
    SUPABASE_KEY=your_supabase_anon_key
    ```

    Start the server:

    <CodeGroup>
      ```bash python3
      python3 main.py
      ```

      ```bash python
      python main.py
      ```
    </CodeGroup>

    The API will be available at `http://localhost:8000`.

    <Tip>
      Once the server is running, explore the auto-generated interactive API reference at [http://localhost:8000/docs](http://localhost:8000/docs). You can test every endpoint directly from the browser using the Swagger UI.
    </Tip>
  </Step>

  <Step title="Set up the client">
    In a new terminal, navigate to the client directory and install dependencies:

    ```bash
    cd client
    npm install
    ```

    Start the Vite development server:

    ```bash
    npm run dev
    ```

    The frontend will be available at `http://localhost:5173` (or the next available port).
  </Step>

  <Step title="Log in and browse the marketplace">
    1. Open `http://localhost:5173` in your browser.
    2. Enter the email and password for a user in your Supabase project and click **Log In**.
    3. You will land on the **Dashboard** overview. Click **Marketplace** in the sidebar to browse available energy contracts.
    4. Select any contract to view its details — including energy type, price per MWh, delivery window, and carbon intensity.
    5. Click **Add to Portfolio** to save a contract to your personal portfolio.
  </Step>
</Steps>
